C++ Text Input Outpur
This is a simple text file manager created using C++. The program has the ability to open an existing text file, create a new text file, and write text to it.

Features
1. Open an existing text file
2. Create a new text file
3. Write text to a text file
How to Use
1.Clone the repository
2. Compile and run the program using your preferred C++ compiler
3. Choose the desired option: "Open file", "Create file", or "Write to file"
4. Follow the prompts to complete the action
Requirements
1. A C++ compiler
